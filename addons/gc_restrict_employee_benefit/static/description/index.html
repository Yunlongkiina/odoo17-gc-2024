Golden Crop Employee Restrict Benifit App

Block Payment when Employee current month consumption is more than limit amount

Check curent customer consumption infomation and limit amount infomation

Description

Basic Settings

Go to the Settings ->Manage Users


Go to the Point of Sale -> select one session-> goto Settings.


Go to the Contacts -> select contact who is employee


Logics

  • If customer is not selected, go through Odoo 15 POS payment logic
  • If Selected customer limit amount is lesss than 0, go through Odoo 15 POS payment logic
  • If Selected customerdoes not use Goldencrop Employee Pricelist,go through Odoo 15 POS payment logic
  • If customer is selected, current customer has limit amount and limit amount bigger than 0, current month consumption is more than limit amount. Current Payment will be blocked.
